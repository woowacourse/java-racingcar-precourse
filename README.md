# 자동차 경주 게임
---
#### 구현할 기능 목록

1. 경주할 자동차의 초깃값을 설정한다.
    - 자동차의 이름은 쉼표(,)로 구분하여 입력받는다.
    - 이름은 5자 이하만 가능하다.
    
2. 경주를 진행한다.
    - 몇 번의 이동을 할 지 사용자에게 입력받는다.
    - 0부터 9까지의 random값을 구한 후 4이상일 경우 전진한다.
    - 경주가 진행되는 중에 진행 상황을 출력한다. 출력할 때에는 자동차 이름을 같이 출력한다.
    
3. 경주가 완료된 후 우승자를 출력한다.
    - 우승자는 한 명 이상일 수 있다.
