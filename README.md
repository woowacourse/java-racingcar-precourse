## 자동차 경주 게임(Racing Car)

 우아한테크코스 2기 프리코스 2주차 미션(191204~)

#### 1. 규칙

-  각 자동차에 이름 부여(5자 이내)
- 사용자는 몇 번의 이동을 할 것인지 입력
-  주어진 횟수동안 n대의자동차 전진 or 정지, 전진하는 자동차 이름 출력
- 전진하는 조건 : 0~9사이에서 random값을 구한 후 값이 4이상일 경우, 전진, 3이
  하일 경우, 멈춤.

- 자동차경주게임을완료한후 우승자 출력
