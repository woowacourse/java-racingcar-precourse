# 미션 - 자동차 경주 게임

## 기능구현 목록
-   Engine : 난수를 생성한 후, 전진/멈춤을 판단. 
-   Car : 경주에 사용될 Car클래스
-   InputConroller : 사용자 입력을 처리
-   Racing : [Null]
-   Validator : 입력값에 유효성 체크
-   Game : 게임진행을 담당
-   ViewController : 출력을 처리
-   Exception : 사용자 입력 값에 대한 예외 클래스
