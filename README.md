# 우아한 테크코스 프리코스 2주차 < 자동차 경주 게임 >


## 기능 요구 사항 
 주어진 횟수 동안 n대의 자동차는 전진 또는 멈출 수 있다.
 각자동차에이름을부여할수있다.전진하는자동차를출력할때자동차이름을같이출력한다. 
 자동차이름은쉼표(,)를기준으로구분하며이름은5자이하만가능하다.
 사용자는몇번의이동을할것인지를입력할수있어야한다.
 전진하는조건은0에서9사이에서random값을구한후random값이4이상일경우전진하고,3이하의값이면멈춘다. 자동차경주게임을완료한후누가우승했는지를알려준다.우승자는한명이상일수있다.
 

### 구현 기능 목록
 1. Main.java 
 	 프로그램을 실행시킨다.
 2. Car.java
 	Car의 이동 상황을 제어한다.
 	- Car() : 생성자
 	- getLocate() : 객체의 현재 위치 리턴
 	- getName () : 객체의 이름을 리턴
 	- carMove () : 랜덤넘버를 통하여 조건에 만족하면 위치값을 증가시킨다.
 3. PlayGame.java
 	Game을 룰에 따라 진행한다.
 	- play() - 게임을 진행한다.
 	- setCarName() : 사용자에게 입력을 받아 Car 객체 배열을 만든다.
 	- nameVaild() : 입력한 Car 이름에 입력에 오류가 있는지 검증
 	- makeRace() : 검증 된 차량이름을 객체 배열에 각각 배열함 
 	- setGameNum () : 사용자에게 게임 횟수를 입력 받음
 	- gameNumVaild () : 게임 횟수 숫자에 대해 검증
 	- gameResult() : 1회 게임을 진행 할때마다 진행결과를 출력하고 max 값을 저장함 
 	- gameWinner() : max 값을 가진 차에 대하여 출력하고 게임 종료!
