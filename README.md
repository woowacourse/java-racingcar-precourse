
# 미션 - 자동차 경주 게임

## 구현 목록
- [ ] 입력
  - [ ] 경주 자동차 입력받기
  - [ ] 시도 횟수 입력 받기
- [ ] 출력
  - [ ] 경주 자동차 입력 안내 메세지 출력
  - [ ] 시도 횟수 입력 안내 메세지 출력
  - [ ] 실행결과 출력
  - [ ] 최종 우승자 출력
- [ ] 게임 진행
  - [ ] 시도 횟수 결정
    - [ ] 0 ~ 2,147,483,647 범위의 숫자만 입력 가능
  - [ ] 자동차 생성
    - [ ] 자동차 이름은 1 ~ 5 글자로 제한한다.
  - [ ] 0 ~ 9 사이의 무작위 값 생성
  - [ ] 자동차 전진
  - [ ] 우승자 결정
