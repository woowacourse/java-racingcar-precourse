# 우아한 테크 코스 <br> 온라인 과제 2주차 Java Racing Car

## 프로젝트 소개
자동차들이 경주하여, 가장 많이 이동한 우승 차량을 뽑는 게임입니다.  
게임에 참여하는 모든 자동차의 이름을 사용자가 입력합니다.  
사용자가 입력한 횟수 동안 각 자동차는 Random하게 앞으로 이동할 수도 이동하지 않을 수도 있습니다.
 
## 기능 목록
### 1. 자동차 이름 입력, 객체 생성
1. 사용자로부터 자동차 이름을 입력받는다.
2. 자동차 이름 입력에 대한 예외 처리를 한다.
3. 자동차 객체를 생성한다.

### 2. 게임 횟수 입력
1. 사용자로부터 게임 횟수를 입력받는다.
2. 게임 횟수 입력에 대한 예외 처리를 한다.

### 3. 자동차 전진 여부 확인
1. 각 자동차가 0에서 9 사이의 난수를 생성한다.
2. 각 자동차가 난수에 따라 전진하거나 멈춘다. 
4. 모든 자동차가 난수에 따라 전진하거나 멈춘다. 

### 4. 자동차 위치 정보 출력
1. 각 자동차의 이름과 현재 위치를 출력한다.
2. 모든 자동차의 결과를 출력한다.

### 5. 게임 최종 승자 결정
1. 각 자동차의 전진 횟수를 토대로 승자를 찾는다.
2. 승자를 출력한다.
