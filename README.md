# 미션 - 자동차 경주 게임

---

## 구현할 기능 목록

- 경주 할 자동차 이름을 묻는 기능
    - 이름은 쉼표(,) 기준으로 구분, 각 이름은 5자 이하
- 이동을 시도할 횟수를 묻는 기능
    - 정수
- 각 이동 차수마다 자동차별 이동거리 계산
- 각 이동 차수마다 자동차별 이동거리 출력
- 모든 이동 완료 후 최종 우승자 선정
- 최종 우승자 출력
