#project name : java racingcar precourse

우아한 프리코스 2주차 미션 racing car

<hr />

## 기능 및 예외
Car class
1. 자동차 전진 기능

GameUI class
1. input string을 GameBoard로 전달
    -2개 이상의 car name이 존재
2. 게임 횟수 n을 gameboard로 전달
    -0이상의 값
3. 출력문

GameBoard
1. input을 통해 차 객체를 생성하는 기능
    -5글자 넘어가지 않도록
2. 게임을 진행하는 기능
3. 전진조건 기능 (랜덤 숫자)
4. 우승자선정 기능
