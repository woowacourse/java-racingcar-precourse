* 미션 내용
 - 자동차 경주 게임
 - 5명 이하의 사람이 랜덤한 숫자를 뽑아서 전진 또는 정지
 - 시도 횟수만큼 반복하고, 최종 승자를 찾는다.
 - 이 때 승자는 여러 명일 수 있다.
 
 * 기능 구현
 1. Car class
  - 자동차 이름의 변수로 객체 생성
  - 자동차 전진
  - 자동차의 전진, 정지 파악 후 위치 이동
  - 자동차 위치 반환
  - 자동차의 이름 반환
 
 2. test class
  - 자동차 이름을 쉼표로 구분해서 받기
  - 시행횟수 받기
  - 받은 시행횟수만큼 게임 진행
  - 자동차의 위치 함수
  - 최종 결과 리스트로 만들어 반환
  - 최종 결과 출력
  
  - 게임 실행 순서에 따른 메인문
