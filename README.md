# 우아한 테크코스 프리코스 2주차 < 자동차 경주 게임 >


## 기능 요구 사항 
   - 주어진 횟수 동안 n대의 자동차는 전진 또는 멈출 수 있다.
   - 각 자동차에 이름을 부여 할 수 있다. 전진하는 자동차를 출력 할 때 자동차 이름을 같이 출력한다.
   - 자동차 이름은 쉼표(,)를 기준으로 구분하며 이름은 5자 이하만 가능하다.
   - 사용자는 몇번의 이동을 할 것인지를 입력 할 수 있어야 한다.
   - 전진하는 조건은 0에서 9사이에서 random값을 구한 후 random 값이 4이상 일 경우 전진하고, 3이하의 값이면 멈춘다.
   - 자동차 경주게임을 완료한 후 누가 우승 했는지를 알려준다. 우승자는 한명 이상일 수 있다.


### 구현 기능 목록
 1. Main.java 
 	 프로그램을 실행시킨다.
 2. Car.java
 	Car의 이동 상황을 제어한다.
 	- Car() : 생성자
 	- getLocate() : 객체의 현재 위치 리턴
 	- getName () : 객체의 이름을 리턴
 	- isCarMove () : 랜덤넘버를 통하여 조건에 만족하면 위치값을 증가시킨다.
 	- createRanNum() : 랜덤 넘버를 생성한다.
 	- oneRace() : 레이스를 1회 진행시킨다.
 	- RoundResult() : 1회 진행 후 결과를 출력한다.

 3. PlayGame.java
 	Game을 룰에 따라 진행한다.
 	- play() - 게임을 진행한다.
 	- getCarName() : 사용자에게 입력을 받아 Car 객체 배열을 만든다.
 	- nameVaild() : 입력한 Car 이름에 입력에 오류가 있는지 검증
 	- makeRace() : 검증 된 차량이름을 객체 배열에 각각 배열한다.
 	- raceStart() : 입력 된 모든 차량에게 경기를 진행한다.
 	- gameWinner() : max 값을 가진 차에 대하여 출력하고 게임 종료!
 	- getMaxPosition() : 모든 차량중에 가장 높은 position 을 가지고 있는 차량의 값을 저장한다.
